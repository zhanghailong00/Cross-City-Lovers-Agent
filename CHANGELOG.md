# Changelog

## v0.1.0 - MVP v1

首个可演示版本，完成最小闭环能力。

### 新增

- 初始化项目目录结构
- 新增 FastAPI 接口入口
- 新增 LangGraph 工作流骨架
- 新增意图解析、候选城市生成、交通方案、配对、评分等服务骨架
- 新增基础领域模型与响应结构
- 新增城市画像和交通估算种子数据
- 新增本地启动脚本与依赖初始化脚本
- 新增最小自动化测试
- 新增用户可读展示字段 `display_text`

### 当前特性

- 支持通过自然语言生成异地情侣汇合推荐结果
- 支持返回推荐城市、备选城市、双方去程方案和推荐理由
- 支持本地接口演示

### 已知限制

- 当前为估算数据，不是实时票务数据
- 当前未接入真实 LLM 主流程
- 当前未实现返程、住宿和完整行程规划
